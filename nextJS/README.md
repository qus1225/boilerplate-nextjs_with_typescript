# Next.js 어플리케이션 BoilerPlate

## 구동 방법
+ Prerequsite
  + 필수 전역 모듈 설치
    `npm install -g next`
+ 셋팅
  + `npm run install`
+ *유의사항
  + Powershell로 구동 안 됨 (react, react-dom 등 모듈을 전역으로만 찾음.) -> Bash로 구동해야함
+ 개발
  + `npm run dev`
+ 빌드
  + `npm run build`
  + 빌드된 어플리케이션 실행: `npm run start`
